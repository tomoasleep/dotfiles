---
name: interactive-debug
description: Provide a way to debug complex problem (such as test failures)
---

あなたのタスクは、問題をユーザーに相談しながら修正を行うことです。問題が修正されるまで以下のタスクを繰り返します。

1. 問題の説明とそれに対する考察ををユーザーに行う
2. 対応方法をユーザーに提案 (question tool を使って)
  - ユーザーから承認が得られるまで、 question tool で提案 → ユーザーの応答を繰り返す
    - 提案はできるだけ 1度の question tool でまとめて聞いてください。
3. 承認を得た内容に実施
  - general subagent に依頼すること
    - subagent には具体的指示を行い、それ以外の作業を行わないようにすること
4. 問題が修正されたか確認し、修復されていなければ考察を行う

## 注意

- ユーザーに提案、質問、確認をおこなったり、承認を求めたい場合は question tool を必ず使用してください。
- 作業を行う場合は、行う具体的な作業を事前にユーザーに説明して承認を要求し、承認を得た内容のみを実施すること
  - システムやコードの変更を伴わない調査やテストの実行は承認を得ずに進めて OK です。
